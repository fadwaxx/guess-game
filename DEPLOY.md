# رفع اللعبة على استضافة Node.js

تم تجهيز المشروع ليعمل كخدمة واحدة: الواجهة React والخادم Socket.IO على نفس الرابط.

## إعدادات الاستضافة

- Root Directory: اتركه على جذر المشروع.
- Build Command: `npm run build`
- Start Command: `npm start`
- Node.js: الإصدار 20 أو أحدث.
- متغير `PORT`: لا تضفه يدويًا إذا كانت الاستضافة تضيفه تلقائيًا.
- متغير `CLIENT_ORIGIN`: غير مطلوب عند تشغيل الموقع والخادم على الرابط نفسه.

## التجربة على الكمبيوتر

افتح نافذتين للأوامر:

1. داخل `server`: نفّذ `npm install` ثم `npm run dev`.
2. داخل `client`: نفّذ `npm install` ثم `npm run dev`.

النسخة لا تحتاج قاعدة بيانات؛ الغرف والنقاط والبطاقات تحفظ مؤقتًا حتى مغادرة الغرفة أو إعادة تشغيل الخادم.
